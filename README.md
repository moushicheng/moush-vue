# moush-vue
在react、vue等MVVM框架如此流行的今天，你要问我仿写一个Vue有什么意义吗?我只能很遗憾的告诉你，没什么意义。但于我个人而言，这能帮助我理解vue的源码原理等等其实就已
经足够了。


## install
> npm run install
## start
> npm run dev

## feature

1. 框架直接操作真实dom而没有用到vdom
2. 只实现了vue的部分功能，因为事无巨细全部实现的话有点搬砖。
